# WebSnapse Backend

### Prerequisites

- Python 3.9.6 or higher
- Pydantic
- Numpy
- FastAPI
- Uvicorn

### Installation

1. Clone the repo
   ```sh
   git clone
   ```
2. Install Python packages
   ```sh
   pip install -r requirements.txt
   ```
3. Run the server
   ```sh
   uvicorn main:app --reload
   ```
4. Open the browser and go to http://localhost:8000/

### Testing

To test, run the following command:

```sh
python tests.py
```
